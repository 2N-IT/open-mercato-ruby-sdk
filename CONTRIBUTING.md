# Contributing to Open Mercato Ruby SDK

## Development Setup

```bash
git clone https://github.com/open-mercato/ruby-sdk
cd open-mercato-ruby-sdk
bundle install
```

## Running Tests

```bash
bundle exec rake spec
```

## Running the Linter

```bash
bundle exec rubocop
```

## Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes and add tests
4. Ensure all tests pass: `bundle exec rake spec`
5. Ensure linting passes: `bundle exec rubocop`
6. Commit your changes
7. Open a Pull Request

## Reporting Issues

Please use [GitHub Issues](https://github.com/open-mercato/ruby-sdk/issues).
